Политика конфиденциальности Последнее обновление: [10.08.2025 г.] 1. Общие положения Настоящая Политика конфиденциальности описывает, как мы обрабатываем данные пользователей в нашем веб-приложении [Калькулятор доставки], размещённом на GitHub Pages. Используя это приложение, вы соглашаетесь с условиями данной политики. 2. Какие данные мы собираем и храним? 2.1. Данные для входа Приложение использует логин и проверочный код, которые хранятся в файле [auth.json] в репозитории GitHub. Эти данные не являются персональными (не привязаны к реальным лицам) и нужны только для доступа к приложению. 2.2. Локальное хранение (localStorage) Приложение сохраняет в вашем браузере: - Историю расчетов, - Настройки (например, выбранный тип курьера), - Рейтинг и другие пользовательские данные. Эти данные не передаются нам и хранятся только на вашем устройстве. 2.3. Подписки через Telegram При нажатии кнопку "Купить" вы переходите в Telegram, где можете связаться с нами для оплаты. Мы не храним ваши банковские реквизиты — платёж обрабатывается вручную через перевод на карту. 3. Как мы используем данные? - Логины и коды — только для входа в приложение. - Данные в localStorage — для работы калькулятора и сохранения ваших настроек. - Сообщения в Telegram — только для оформления подписки. Мы не передаём ваши данные третьим лицам и не используем их для рекламы или аналитики. 4. Безопасность данных - Логины и коды хранятся в открытом JSON-файле на GitHub (без доступа к персональным данным). - Данные в localStorage защищены стандартными механизмами браузера. - Telegram-переписка удаляется после обработки запроса. 5. Ваши права Вы можете: - Очистить localStorage (данные удалятся без возможности восстановления), - Запросить удаление логина из JSON-файла (написать в Telegram), - Отказаться от подписки в любой момент. 6. Изменения в политике Мы можем обновлять этот текст. Актуальная версия всегда будет доступна по этой ссылке. 7. Контакты По вопросам конфиденциальности пишите: [@PCH1s].
